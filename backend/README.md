# Portfolio Backend (Express + TypeScript + MongoDB)

## Folder structure
See `portfolio-app/backend/src`.

## Setup
```bash
cd portfolio-app/backend
npm install
```

## Environment
Create `portfolio-app/backend/.env` from `.env.example`.

## Run
### Dev
```bash
npm run dev
```

### Build
```bash
npm run build
```

### Start
```bash
npm run start
```

## APIs
- `POST /api/auth/login`
- `GET /api/projects`
- `GET /api/projects/:slug`
- `POST /api/projects`
- `PUT /api/projects/:id`
- `DELETE /api/projects/:id`
- `GET /api/blogs`
- `GET /api/blogs/:slug`
- `POST /api/blogs`
- `PUT /api/blogs/:id`
- `DELETE /api/blogs/:id`
- `GET /api/github`
- `POST /api/contact`
